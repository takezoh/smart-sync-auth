Smart Sync for Obsidian

Bidirectional sync between your Obsidian vault and Google Drive.

GitHub Install

Features

  • Bidirectional sync — changes flow both ways automatically
  • Three-way merge for concurrent edits
  • Six conflict resolution strategies
  • Incremental sync using Google Drive change tracking
  • Works on desktop and mobile
  • Open source (MIT License)

How it works

Smart Sync compares your local vault, remote Google Drive folder, and the last known sync state to determine what changed and where. It then applies the minimal set of operations to bring both sides in sync. All processing happens locally on your device — no intermediary servers.
